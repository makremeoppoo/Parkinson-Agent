End-to-end user flows
🩺 Doctor flow
1 Register / sign in with email + password
2 Add patient with a code (P-1042 …)
3 Backend generates unique linkToken → stores in Firestore
4 Copy link → share with patient
5 View all analyses + HTML report per patient
6 "Scan Now" — in-office scan saved to patient's data
🧑‍⚕️ Patient flow
1 Open unique link ?pt=<token> — no login needed
2 App validates token → GET /patient-session/:token
3 Choose 👋 Hand Scan (7s) or 🧍 Full Body (10s)
4 POST /patient-analyze — token + scanMode → Gemini → Firestore
5 Severity scores + voice feedback displayed
6 History + My Report — all via linkToken, no account
