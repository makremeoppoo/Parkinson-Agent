End-to-end user flows
🩺 Doctor flow
① Sign in with email + password (Firebase Auth)
② DoctorDashboard → add patient code → copy scan link
③ Choose 👋 Hand Scan (7 s) or 🧍 Full Body (10 s)
④ POST /analyze-frame + Firebase JWT + scanMode → Gemini
⑤ Results saved to Firestore · TTS reads voice_message
⑥ REPORT → self-contained HTML with Chart.js history
🧑‍⚕️ Patient flow (home)
① Receives link: https://app.url/?pt=<token>
② App validates token → GET /patient-session/:token
③ Choose 👋 Hand Scan (7 s) or 🧍 Full Body (10 s)
④ POST /patient-analyze — token + scanMode → Gemini
⑤ Results in Firestore · TTS speaks diagnosis aloud
⑥ History + My Report — all via linkToken, no account
